## [1.0.0] - [Unreleased]

### Comment
- Initial release

### Added
- Hunt log import and analysis with detailed session breakdown
- Hunt summary dialog with XP/h, loot/h and configurable XP modifier
- Progress view with level forecast based on current hunt cadence
- Economy tracker: sales logging and recent sales overview
- Character import via TibiaData API with full character graph (badges, achievements, houses, deaths)
- Character switching and persistent storage via SQLite
- Date range picker for filtering hunt sessions
- Auto-updater: automatic update check, download, and installation (Windows, Linux, macOS)
- Localization: German, Spanish, Polish, Portuguese
- Crash diagnostics with zip export
